Our Unity-based, ROS-compatible simulator is fully integrated with the popular open source platforms Baidu Apollo and Tier IV Autoware. Our source code is on Github and free to try.

Github

Features

  • Full Apollo and Autoware integration

  • ROS/ROS2 integration

  • Real world sensor models including LiDAR, RADAR, GPS, IMU, camera

  • HD map support and HD map editing capability

  • Ground truth 2D/3D obstacles export
